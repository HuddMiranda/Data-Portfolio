# SQL Data Analysis Portfolio

## Sobre o projeto

Este repositório reúne consultas SQL desenvolvidas para análise e transformação de dados.

Os scripts demonstram técnicas utilizadas em projetos de análise de dados e business intelligence, como:

- Agregação de dados
- Transformação de informações
- Geração de relatórios analíticos
- Criação de colunas dinâmicas
- Organização de dados para análise

## Tecnologias utilizadas

- SQL
- PostgreSQL
- Transformação de dados
- Consultas analíticas

## Estrutura do repositório

SQL

01_Pivot_Dinâmico.sql  
Geração dinâmica de colunas utilizando metadados da base de dados.

02_Tratamento_Dados_Imóveis.sql  
Transformação de dados cadastrais de imóveis em um dataset estruturado para análise.

03_Dívida_Por_Contribuinte.sql   
Relatório consolidado do valor total de dívida por contribuinte.

04_Dívida_Por_Título.sql  
Consulta que identifica os maiores créditos individuais registrados no sistema.
